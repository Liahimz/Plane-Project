Установка и запуск программы (Ubuntu / Mac osx (HighSierra и ниже)) : 

	Установка библиотек Ubuntu: sudo apt install libglfw3-dev
	Установка библиотек Mac: brew install glfw

	Запуск :

		mkdir build
		cd build
		cmake ..
		make 
		./main -planescene
